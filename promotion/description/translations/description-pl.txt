# Application:
OI Lista Zakupów

# Promo text:
OI Lista Zakupów pozwala śledzić listę zakupów. Można jej używać również dla innych typów list: lista rzeczy do zrobienia lub lista gości.

# Description:
Lista zakupów Ol to potężna aplikacja, która umożliwia łatwe tworzenie i zarządzanie listami kontrolnymi. Lista zakupów OI ułatwia dodawanie elementów do listy, zaznacz element, usuń go z listy. Aplikacja może śledzić oddzielne cen dla każdego sklepu, którego używasz, i oferuje wysoki poziom personalizacji. Rozmiar czcionki, porządek sortowania, kolumny listy, a zachowanie podczas czyszczenia listy są wszystkie ustawienia kontrolowane przez użytkownika. Wszystkie te funkcje są połączone w wyraźny układ, który można dostosować do różnych motywów.

Cechy: 
* Dodawaj, edytuj i usuwaj listy.
* Tworzenie, usuwanie i sprawdzanie pozycji szybko i łatwo.
* Różnorodność tematów

* Śledź oddzielnie ceny dla każdego sklepu.
* Dostosuj kolumny listy.

Dostępne rozszerzenia do "Listy zakupów Ol": 
* Add a recorded message to a note with "OI Voice Notes".
* Import/export shopping lists with "OI Convert CSV".
* Pokaż informacje licencyjne z "O Ol".
* Enter/check items with "Barcode OI Plugin".

Applications that work with "OI Shopping List": 
* Aktualizuj z "OI Uptade".
* Backup lists with "My Backup Pro".
* Import ingredients of recipes with "Bites".
